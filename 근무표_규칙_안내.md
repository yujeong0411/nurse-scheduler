# 근무표 자동생성 규칙 안내

> 시스템이 근무표를 어떻게 만드는지, 어떤 규칙이 적용되는지 설명합니다.

---

## 1. 신청 코드 종류

| 분류 | 코드 |
| :--- | :--- |
| **근무** | D, E, N, D9, D1, 중1, 중2 |
| **휴무** | OFF, 주, 법휴, 수면, 생휴, 휴가, 특휴, 공가, 경가, 보수, 병가, POFF, 필수, 번표 |
| **제외** | D 제외, E 제외, N 제외 |
| **OR** | `/` 구분자 (예: `D/휴가` → 둘 중 하나 배정) |

### 근무 신청 상세
| 코드 | 설명 |
|------|------|
| D | 데이 근무 |
| E | 이브닝 근무 |
| N | 나이트 근무 |
| 중2 | 중간근무 2 (비고1 역할이 '중2'인 간호사만 배정 가능) |
| D9, D1, 중1 | **직접 입력 전용** — 신청 시에만 배정, 시스템이 자동 배치하지 않음 |

### 휴무 신청 상세
| 코드 | Hard/Soft | 설명 |
|------|-----------|------|
| OFF | Soft | 일반 휴무. 정확히 OFF로 배정되어야 반영된 것으로 처리 |
| 주 | Hard | 주휴일 (고정 요일 지정 시 그 날에만 배정) |
| 법휴 | Hard | 법정공휴일 휴무 (공휴일에만 신청 가능) |
| 수면 | Hard (명시 신청 시) | N 근무 누적 시 자동 부여. 직접 신청 시 반드시 해당 날 배정 |
| 생휴 | Soft | 여성 간호사 월 1회 자동 부여 |
| 휴가 | Soft | 연차 휴가 |
| POFF | Hard | 임산부 전용 — 4연속 근무 후 자동 부여 |
| 번표, 병가 | **Hard** | 신청한 수만큼 반드시 배정 |
| 특휴, 공가, 경가, 보수, 필수 | Soft | 가능하면 반영. 신청하지 않은 날에는 자동 배정 없음 |

### OFF 신청 vs 구체적 휴무 신청
| 신청 코드 | 배정 충족 조건 |
| :--- | :--- |
| **OFF** | 정확히 OFF로 배정되어야 충족 (수면·생휴 등 다른 타입은 미충족) |
| **휴가·수면·생휴·특휴 등** | 정확히 그 타입으로 배정되어야 충족 |

> S1 보상 및 assignment_log `is_assigned` 모두 exact match.
> OFF 신청일에 수면·생휴 등이 자동 배정될 수 있으나, 그 경우 신청은 미반영으로 표시됨.

## 2. 신청 조건 (A/B) 및 점수

### 조건 구분
| 조건 | 의미 | 월 제한 |
|------|------|---------|
| **A조건** | 꼭 쉬어야 하는 날, 우선순위 높음 | 월 최대 3개 |
| **B조건** | 가능하면 반영해 달라는 희망 사항 | 제한 없음 |

> A조건이 B조건보다 점수와 관계없이 **항상 먼저** 반영됩니다. (A 최저 800 > B 최고 750)

### 점수 (Score)
- 초기값: **100점** (매월 리셋)
- A 신청 1개 사용 시: **-1점**
- B 신청 1개 사용 시: **-3점**
- OR 신청 (`D/휴가` 형태)은 **1개**로 집계

### 반영 우선순위 가중치

| score | A weight (`800 + score×5`) | B weight (`250 + score×5`) |
|-------|---------------------------|---------------------------|
| 100 | 1300 | 750 |
| 85  | 1225 | 675 |
| 70  | 1150 | 600 |
| 0   | 800  | 250 |

- A 최저(score 0) = 800 > B 최고(score 100) = 750 → **A가 항상 우선**
- score 1점 차이 → weight 5 차이

## 3. Hard 제약 목록

| 번호 | 내용 |
| :--- | :--- |
| **H1** | 하루 1배정 |
| **H2** | 일일 D/E/N 인원 정확히 고정 |
| **H2a** | 중2 role 아닌 간호사 → 중2 불가 |
| **H2b** | D9/D1/중1 → hard 요청 없으면 자동배정 불가 |
| **H3** | 역순 금지 (D → 중2 → E → N 순서만) |
| **H3a** | N→1휴무→D·중간·N 금지 (E만 허용) / N→N 연속(NNN)은 허용 |
| **H3c** | N 다음날 보수/필수/번표 금지 |
| **H4** | 최대 연속 근무 5일 |
| **H5** | 최대 연속 N 3개 |
| **H6** | NN 블록 종료 후 2일 휴무 |
| **H8** | 병가/D9/D1/번표/수면(명시) hard 배정 |
| **H9** | 제외 요청 강제 금지 |
| **H10** | 고정 주휴요일에 주 배정 |
| **H10a** | 주는 고정 주휴일에만 배치 가능 |
| **H10b** | 법휴는 공휴일에만 배치 가능 |
| **H11** | 주당 OFF 정확히 1개 (주4일제 2개). 공휴일 주는 법휴가 대체 가능 (`OFF + 법휴 ≥ 1`, `OFF ≤ 1`) |
| **H12** | 매 근무(D/E/N) 책임 1명 이상 |
| **H13** | 매 근무 책임+서브차지 N명 이상 |
| **H14** | 역할 누적 인원 제한 |
| **H15** | 책임만 1명 이하 |
| **H17** | 임산부 4연속 근무 금지 |
| **H18** | 공휴일 비근무 시 법휴만 허용 |
| **H19** | 임산부 4연속 근무 후 POFF 강제 |
| **H20** | 휴무 편차 ±2 이내 (주4일제 +4) |
| **H21** | 신청 휴무 샌드위치 금지 — d일 휴무 신청 시, d-1·d+1이 모두 휴무이면 d일도 반드시 휴무 |

### N 이후 복귀 패턴 요약

| 패턴 | 허용 여부 |
|------|----------|
| N → N (연속, NNN까지) | 허용 |
| N → 휴무 → E | 가능 |
| N → 휴무 → D/중간/N | **불가** (2일 휴무 필요) |
| N → 휴무 → 휴무 → D/중간 | 가능 |
| NN 이상 종료 → 2일 미만 휴무 후 근무 | **불가** |

| 번호 | 내용 | 가중치 |
| :--- | :--- | :--- |
| **S1** | 희망 요청 반영 (exact match) | A: +800+score×5 / B: +250+score×5 |
| **S1a** | OR 신청 그룹 (exact match) | 동일 공식 — 그 코드 배정 시에만 보상 |
| **S1b** | hard 휴무 사이 빈날 OFF 유도 | +250 |

| **S3** | N 균등 분배 | -8 × (max−min) |
| **S4** | 주말 근무 균등 | -8 × (max−min) |

| **S7** | 연속 휴무 보상 | +15 / 연속휴무쌍 |
| **S8** | 월 N 초과 억제 | -300 / 초과 N 1개 |

## 5. 특수 휴무 자동 배정

### 생휴 (여성 간호사 월 1회)
- **대상:** 여성 간호사 전원
- **조건:** 해당 달에 아직 생휴를 사용하지 않은 경우
- **방식:** '생리휴가 적용' 설정이 켜져 있을 때 자동으로 1회 배정
- **직접 신청 시:** 신청한 날짜로 배정

### 수면 (나이트 누적 시 자동 부여)
- **조건 1 (홀수 달):** 해당 달 N 횟수 ≥ 7회
- **조건 2 (짝수 달):** 전월 N + 이번 달 N 합계 ≥ 11회
- **이월:** 홀수 달에 조건 미충족 시 → 다음 달로 이월 (보류 수면)
- **직접 신청 시:** 이월된 수면을 특정 날짜에 신청하면 반드시 해당 날 배정 (hard)

### POFF (임산부 전용)
- **조건:** 임산부 간호사가 4일 연속 근무한 직후
- **방식:** 4연속 근무 바로 다음 날에 자동 배정

## 6. 전체 우선순위 요약

```
1순위  병가 / 번표 / 수면(명시 신청) / D9 / D1  →  반드시 배정 (Hard)
2순위  D 제외 / E 제외 / N 제외               →  반드시 금지 (Hard)
3순위  A조건 신청                             →  강력 반영 (800 + score×5)
4순위  B조건 신청                             →  반영 (250 + score×5)
5순위  D/E/N/주말 균등 분배                   →  공정성 페널티 (S2~S4)
6순위  N 연속·휴무 연속                        →  패턴 보너스 (S6, S7)
7순위  월 N 초과 억제                          →  페널티 -300/초과 N (S8)
```

> **참고:** 모든 규칙이 동시에 완벽하게 만족되지 않을 수 있습니다.
> 시스템은 전체 간호사의 만족도를 최대화하는 방향으로 최적해를 찾습니다.
