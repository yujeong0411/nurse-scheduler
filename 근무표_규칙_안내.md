# 근무표 자동생성 규칙 안내

> 시스템이 근무표를 어떻게 만드는지, 어떤 규칙이 적용되는지 설명합니다.

---

## 1. 신청 코드 종류

### 근무 신청
| 코드 | 설명 |
|------|------|
| D | 데이 근무 |
| E | 이브닝 근무 |
| N | 나이트 근무 |
| 중2 | 중간근무 2 (지정 역할만 배정 가능) |
| D9, D1, 중1 | **직접 입력 전용** — 신청 시에만 배정, 시스템이 자동 배치하지 않음 |

### 휴무 신청
| 코드 | 설명 |
|------|------|
| OFF | 일반 휴무 — 어떤 휴무 타입으로 배정되어도 반영된 것으로 처리 (생휴·수면 자동배정 포함) |
| 주 | 주휴일 (고정 요일 지정 시 그 날에만 배정) |
| 법휴 | 법정공휴일 휴무 (공휴일에만 신청 가능) |
| 수면 | N 근무 누적 시 자동 부여 (아래 참고) |
| 생휴 | 여성 간호사 월 1회 자동 부여 (아래 참고) |
| 휴가 | 연차 휴가 |
| POFF | 임산부 전용 — 4연속 근무 후 자동 부여 |
| 특휴, 공가, 경가, 보수, 필수, 번표, 병가 | 기타 특수 휴무 (신청 수만큼 정확히 배정) |

### 제외 신청
| 코드 | 설명 |
|------|------|
| D 제외 | 해당 날짜 데이 근무 절대 불가 |
| E 제외 | 해당 날짜 이브닝 근무 절대 불가 |
| N 제외 | 해당 날짜 나이트 근무 절대 불가 |

### OR 신청
슬래시(`/`)로 구분하면 **둘 중 하나** 배정 요청이에요.  
예: `D/휴가` → D 또는 휴가 중 가능한 것으로 배정

---

## 2. 신청 조건 (A/B) 및 점수

### 조건 구분
| 조건 | 의미 | 월 제한 |
|------|------|---------|
| **A조건** | 꼭 쉬어야 하는 날, 우선순위 높음 | 월 최대 3개 |
| **B조건** | 가능하면 반영해 달라는 희망 사항 | 제한 없음 |

> A조건이 B조건보다 점수와 관계없이 **항상 먼저** 반영됩니다.

### 점수 (Score)
- **초기값: 100점** (매월 리셋)
- A 신청 1개 사용 시: **-1점**
- B 신청 1개 사용 시: **-3점**

점수가 높을수록 같은 조건 내에서 먼저 반영됩니다.

### 반영 우선순위 가중치

| score | A 신청 반영 강도 | B 신청 반영 강도 |
|-------|-----------------|-----------------|
| 100점 | 1300 | 650 |
| 85점 | 1225 | 575 |
| 70점 | 1150 | 500 |
| 0점 | 800 | 150 |

> **A 신청은 score 0점이어도 (800) > B 신청 score 100점 (650)** → A조건이 항상 우선

## 3. 반드시 지켜지는 규칙

아래 규칙들은 **어떤 신청이 있어도 절대 어길 수 없는** 기본 규칙입니다.

### 기본 배정 규칙
| 규칙 | 내용 |
|------|------|
| 하루 1배정 | 하루에 근무 또는 휴무 중 하나만 배정 |
| 일일 인원 고정 | D/E/N 각 근무는 매일 정해진 인원수 정확히 유지 |
| 역순 근무 금지 | 근무 순서는 **D → 중2 → E → N** 방향만 가능 (예: N 다음날 D 불가) |

### N(나이트) 관련 규칙

N 이후 복귀 패턴 요약:

| 패턴 | 허용 여부 |
|------|----------|
| N → N (연속) | 허용 (NNN까지 가능) |
| N → E (바로 다음날) | **불가** (역순 금지) |
| N → 휴무 → E | **가능** |
| N → 휴무 → D/중간/N | **불가** (2일 휴무 필요) |
| N → 휴무 → 휴무 → D/중간 | 가능 |
| NN 이상 종료 → 2일 휴무 | **필수** |

| 규칙 | 내용 |
|------|------|
| N 직후 제한 | N 바로 다음날: N 연속 또는 휴무만 가능 (D/E/중간 불가) |
| N 후 1일 휴무 다음날 | E만 허용 — D/N/중간 불가 (최소 2일 휴무 필요) |
| NN 이상 블록 종료 후 | 반드시 2일 이상 휴무 |
| 최대 연속 N | 3개 연속까지 허용 |
| N 다음날 추가 금지 | 보수/필수/번표는 N 바로 다음날 배정 불가 |
| 월 N 권고 | 월 최대 6개 권고 (인원 부족 시 초과 허용) |

### 연속 근무 규칙
| 규칙 | 내용 |
|------|------|
| 최대 연속 근무 | 5일 연속 근무까지만 허용 |

### 주휴/공휴일 규칙
| 규칙 | 내용 |
|------|------|
| 고정 주휴일 | 지정된 요일에 반드시 주휴 배정 |
| 주휴는 지정 요일에만 | 지정 요일 외에는 주휴 배정 불가 |
| 법휴는 공휴일에만 | 공휴일이 아닌 날에는 법휴 배정 불가 |
| 주당 OFF | 매 주 정확히 OFF 1개 (주4일제는 2개) |

### 등급/역할 규칙
| 규칙 | 내용 |
|------|------|
| 책임 간호사 | 매 근무(D/E/N)에 책임 간호사 최소 1명 |
| 책임+서브차지 | 매 근무에 책임+서브차지 합산 최소 N명 |
| 책임만 1명 이하 권고 | 근무에 책임만 단독 배치는 최대 1명 |
| 역할 누적 제한 | 비고1 역할 티어별 누적 인원 제한 준수 |
| 중2 역할 제한 | 비고1이 '중2'인 간호사만 중2 배정 가능 |

### 임산부 규칙
| 규칙 | 내용 |
|------|------|
| 4연속 근무 금지 | 임산부는 4일 연속 근무 불가 |
| POFF 자동 부여 | 4연속 근무 직후 반드시 POFF 추가 |

### 공휴일 근무 규칙
공휴일에 근무하지 않는 간호사는 **법휴만 배정 가능** (일반 OFF 불가)

### 휴무 편차 제한
전체 간호사 간 총 휴무일 수 편차가 **±2일 이내**로 유지됩니다.  
(주4일제 간호사는 +4일 여유 추가)

## 4. 최대한 반영되는 규칙

아래 규칙들은 **가능하면 지키되, 불가피할 경우 절충**되는 규칙입니다.

| 규칙 | 내용 | 효과 |
|------|------|------|
| **신청 반영** | 간호사 희망 신청 최대한 반영 | A > B, score 높을수록 우선 |
| **OR 신청 반영** | D/휴가처럼 둘 중 하나 반영 | 동일 가중치 |
| **휴무 사이 빈날** | 하드 휴무 사이에 낀 날은 OFF로 유도 | +250 보너스 |
| **D/E/N 균등** | 간호사 간 D/E/N 횟수 균등하게 분배 | 불균형 시 페널티 |
| **N 균등** | 나이트 횟수 균등 분배 | 불균형 시 페널티 |
| **주말 근무 균등** | 주말 근무 횟수 균등 분배 | 불균형 시 페널티 |
| **일반 간호사 권고** | 비등급 간호사 한 근무에 3명 이하 권고 | 초과 시 페널티 |
| **N 연속 블록** | NNN처럼 N을 연속으로 묶어 배치 | 연속 쌍당 보너스 |
| **연속 휴무 유도** | 휴무가 뭉쳐서 오도록 배치 | 연속 쌍당 보너스 |
| **월 N 초과 억제** | N 6개 초과 시 강력 페널티 | 인원 부족 시만 초과 허용 |

## 5. 특수 휴무 자동 배정

별도 신청 없이도 자동으로 배정되는 휴무가 있습니다.

### 생휴 (여성 간호사 월 1회)
- **대상:** 여성 간호사 전원
- **조건:** 해당 달에 아직 생휴를 사용하지 않은 경우
- **방식:** 관리자가 '생리휴가 적용' 설정 켜져 있을 때 자동으로 1회 배정
- **직접 신청한 경우:** 신청한 날짜로 배정

### 수면 (나이트 누적 시 자동 부여)
- **조건 1 (홀수 달):** 해당 달 N 횟수 ≥ 7회
- **조건 2 (짝수 달):** 전월 N + 이번 달 N 합계 ≥ 11회
- **이월:** 홀수 달에 조건 미충족 시 → 다음 달로 이월 (보류 수면)
- **방식:** 조건 충족 시 시스템이 자동으로 수면 1일 배정

### POFF (임산부 전용)
- **조건:** 임산부 간호사가 4일 연속 근무한 직후
- **방식:** 4연속 근무 바로 다음 날에 자동 배정

## 6. 배정 우선순위 요약

```
1순위  병가 / D9 / D1       → 반드시 배정 (절대 규칙)
2순위  제외 요청             → 반드시 제외 (절대 규칙)
3순위  A조건 신청            → 강력 반영 (score 높을수록 우선)
4순위  B조건 신청            → 반영 (score 높을수록 우선)
5순위  D/E/N/주말 균등 분배  → 공정성 페널티
6순위  N 연속·휴무 연속      → 패턴 보너스
7순위  월 N 초과 억제        → 강한 페널티 (-300/초과 개수)
```

---

> **참고:** 모든 규칙이 동시에 완벽하게 만족되지 않을 수 있습니다.  
> 시스템은 전체 간호사의 만족도를 최대화하는 방향으로 최적해를 찾습니다.
